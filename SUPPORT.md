# Support

## Getting help

Use the following paths depending on what you need:

| Need | Best place |
| --- | --- |
| Bug report | GitHub Issues |
| Feature request | GitHub Issues |
| Usage question | GitHub Issues with context and command output |
| Security report | GitHub Security Advisories |
| Contribution guidance | [CONTRIBUTING.md](./CONTRIBUTING.md) |

## Before opening an issue

Please check:

1. [README.md](./README.md)
2. [QUICKSTART.md](./QUICKSTART.md)
3. [USER_GUIDE.md](./USER_GUIDE.md)
4. [TESTING.md](./TESTING.md)

When reporting a problem, include:

- your operating system
- how `tk` was installed
- the command you ran
- the exact error output
- your server URL / remote name (from `tk status`)

## Project status

This project is actively developed, but support is best-effort. Clear,
reproducible reports get the fastest response.
